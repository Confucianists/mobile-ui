# mobile-ui
移动端app混合开发框架，vue+apicloud，es6，stylus，使用vw布局，兼容android 4.4和iOS8及以上。

包含功能代码：
  1.css公共样式。css公共函数。（stylus编写）
  2.js公共utils函数。
  3.app版本更新。
  4.axios。promise。
  5.路由跳转、页面切换动画。
  6.基础滴滴cube-ui二次封装相关组建。
  7.原生功能，直接通过js调用apicloud的原生模块。

运行项目：
  克隆项目到本地---> npm install安装依赖包 ---> npm run dev运行项目
